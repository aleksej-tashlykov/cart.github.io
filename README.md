<h1 align="center">Корзина для интернет-магазина</h1>

<h2 align="center">Как работает корзина</h2>
<p>Выберите товар и нажмите на кнопку "Добавить в корзину". Товар появится в корзине. Нажмите на кнопку "Добавить в корзину" еще раз чтобы увеличить количество товара на единицу. Кнопка <img src="https://www.iconfinder.com/icons/211863/minus_round_icon" alt="Уменьшить на 1"/> уменьшает количество товара на единицу. Кнопка &#215; удаляет товар из корзины.</p>
